# 📑 Todo List 사이트

## 📖 프로젝트 소개
- 사용자가 할 일을 등록하고 완료할 수 있는 `Todo List` 사이트입니다.

## 🖥️ 기능
- 할 일 등록
- 할 일 수정
- 할 일 삭제
- 할 일 완료
- 전체 할 일 삭제
- 남은 할 일의 수 확인
- 전체 할 일 / 해야 할 일/ 완료한 일 확인
